basit bir portföy HTML şablonu. Bu şablon, başlık (header), altlık (footer) ve içerik (article) kısmını içerir. Örnek çalışmalarımızı görseller ve sütunlar kullanılarak yapıladırılsın.  sayfa responsive olsun. fotoğrafların yanında metin yazabileceğim bir box ekle. 'hakkında' kısmı ekle ve ortala. ayrıca menu ekle.




<!DOCTYPE html>

        }

            justify-content: center;

            margin-top: 10px;
        }

            text-align: center;

        }

            background-color: #444;
            color: #fff;
            padding: 10px;
            text-align: center;
        }

        }
    </style>
</head>
